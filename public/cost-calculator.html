Assumptions — per-tier journey (advanced)
Each deal runs: a research tool-loop, then 3 Director rounds (critique → revise) — fixed. The numbers below define each tier's journey and feed the costs above. Shared: cached system prefix 12,000 tok (author) / 6,000 tok (Director).
How a deal is priced. Research loop: the agent re-sends its growing transcript each turn, so without caching it re-pays for the stable prefix every turn — that quadratic re-read is the worst-case driver, and prompt caching is what collapses it. Then 3 fixed Director rounds: each round the Director critiques the draft and the author revises it. The coherence gate runs in code at $0. Pipeline per NEW_DEAL.md / HEADLESS.md. Tiers reflect research difficulty: a well-documented filer (SpaceX) converges fast; a thinly-covered private name needs far more searching and reconciliation.
